Safari Books Online allows you to read many of the books from O'Reilly (http://www.oreilly.com/) for free if you use a UCSC computer.

From on campus:
http://proquest.safaribooksonline.com/

If you are not on campus, you must first authenticate:
http://proquest.safaribooksonline.com.oca.ucsc.edu

The library has further information at:
http://library.ucsc.edu/services/off-campus-access-faq

Once you have accessed the Safari web page, you can search for:

  • Learning Gnu Emacs
  • Learning the Bash Shell
  • Learning the Vi and Vim Editors
  • Linux in a Nutshell
  • Managing Projects with Gnu Make
